# goonbot5

**goonbot** is a swiss-army-knife bot used in my private discord server.

### Powered by
- Pycord (Discord API)
- Raspberry Pi (Host)
- Rich (Terminal styling)
- twitch-python (Twitch Integration)
- python-youtube (YouTube Integration)
